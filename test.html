Chrome 擴充套件部署

本頁面用於透過 GPO 發佈 Google Chrome 擴充功能。

📦 擴充檔案列表：

🔗 點我下載 your-extension.crx

🔗 檢視 update.xml

🛠 GPO 設定參考：

請在 GPO 中的 ExtensionInstallForcelist 新增下列條目：

xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx;https://yourusername.github.io/your-repo/update.xml

其中前面為擴充功能 ID，後面為 GitHub Pages 上的 XML 檔案網址。

📘 備註：

  • 請確認網址為 https，否則 Chrome 將拒絕安裝。
  • 不要放置敏感資料於此頁面，因為是公開可見的。
  • 如需更新擴充，請重新打包 .crx 並更新 update.xml 的版本號。
